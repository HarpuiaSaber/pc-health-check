# Windows 11 Compatibility Checker (Java FX Demo)

This application is a specialized demo designed to mimic the **Microsoft PC Health Check** experience. It provides a quick and intuitive way to see if your computer is ready for the next generation of Windows.

## 📋 Requirements

* **Operating System:** Windows 10 or 11.
* **Java Runtime:** Java 21.

## 📄 Note
This project is an independent functional demo. It is intended to showcase UI/UX design and system integration capabilities and is not an official product of Microsoft Corporation.
